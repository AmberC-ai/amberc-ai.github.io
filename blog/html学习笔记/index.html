HTML学习笔记

HTML简介

HTML是一种超文本标记语言，使用标记标签来描述网页。

一般使用专业的HTML编辑器来编辑HTML，主要包括：

  • Adobe Dreamweaver

  • Microsoft Expression Web

  • CoffeCup HTML Editor

也可使用文本编辑器如：Notepad(PC)、TextEdit(Mac)来学习HTML

HTML基本格式

  <!DOCTYPE HTML>
  <html>
  <head>
      <meta>
      <title></title>
      </head>
  <body>
  <h1>第一个标题</h1>
  <p>第一个段落</p>
  </body>
  <html>

!DOCTYPE用来声明html的版本类型，不同版本的html有不同的声明，详情见百度。元素中的内容为页面可见内容

属性

属性提供了有关HTML元素的更多信息，总是在开始标签中规定。如：

  <html>
      <body>
          <div name="Amber">
              块
          </div>
      </body>
  </html>

name属性为元素定义名称

注：属性总是以名称/值对的形式出现，如name=Amber,name是属性，Amber是属性值。

属性值始终被包含在引号内，通常为双引号。但有时属性值里本身就包含双引号，这时必须使用单引号。如：

  name='Bill"HelloWorld"Gates'

更多属性标签详见https://www.w3school.com.cn/tags/html_ref_standardattributes.asp

HTML块级元素

HTML块级元素（block level element)在显示时，通常会以新行开始。h1、p、table、ul等都是块级元素。

div元素

div元素没有特定的含义，但可作为其他元素的容器，再用样式对页面进行布局的时候，可以对较大的块进行布局；div元素可以通过设置板块对页面进行布局。

HTML内联元素

与块级元素相反，内联元素通常不会以新行开始。b(粗体)、td、a、img等都是内联元素。

span元素

与div相同，span元素也没有特定的含义，与css同用时，可对内联元素进行设定。

HTML 样式

style属性提供了一种能改变所有HTML元素样式的方法。通过 HTML 样式，能够通过使用 style 属性直接将样式添加到 HTML 元素。

外部样式表

link标签定义文档与外部资源的关系。当样式需要被应用到很多页面的时候，外部样式表将是理想的选择。使用外部样式表，你就可以通过更改一个文件来改变整个站点的外观。

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

内部样式表

当单个文件需要特别样式时，就可以使用内部样式表。你可以在 head 部分通过 style标签定义内部样式表。

<head>

<style type="text/css">
body {background-color: red}
p {margin-left: 20px}
</style>
</head>

内联样式

<html>
    <body>
        <h1 style="bacground-color:red">
            标题背景颜色为红色
        </h1>
        <h1 style="text-align:center">
            标题居中对齐
        </h1>
        <p style="font-family:italic">
            段落字体为斜体
        </p>
        <p style="color:yellow">
            段落文字为黄色
        </p>
        <p style="font-size:20px">
            段落文字大小为20像素
        </p>
    </body>
</html>

其中

  • background-color属性定义了背景颜色
  • text-align属性定义了对齐方式
  • font-family属性定义了字体
  • color属性定义了字体颜色
  • font-size属性定义了字体大小

HTML4引入style后也废除了若干标签和属性，编写时不建议使用。具体废除标签和属性详见百度

HTML字符实体

在 HTML 中，某些字符是预留的。如：在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签。如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用字符实体（character entities）。

详细的字符实体参考手册见网址https://www.w3school.com.cn/tags/html_ref_entities.html

HTMLmeta元素

meta 元素被用于规定页面的描述、关键词、文档的作者、最后修改时间以及其他元数据。

元数据可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。

<meta name="description" content="Free Web tutorials on HTML, CSS, XML" />

上面这串代码定义页面的描述

<meta name="keywords" content="HTML, CSS, XML" />

上面这串代码定义页面的关键词

HTML文件路径

绝对路径

绝对文件路径是指向一个因特网文件的完整 URL：

<img src="https://www.w3school.com.cn/images/picture.jpg" alt="flower">

相对路径

src=“picture.jpg” picture.jpg 位于与当前网页相同的文件夹
src=“images/picture.jpg” picture.jpg 位于当前文件夹的 images 文件夹中
src=“/images/picture.jpg” picture.jpg 当前站点根目录的 images 文件夹中
src=“../picture.jpg” picture.jpg 位于当前文件夹的上一级文件夹中

HTML iframe

iframe标签用于创建网页中的网页

<iframe src="URL" width="200" height="200" frameborder="0"></iframe>

src属性说明内网页的地址，width、height属性规定内网页的宽高，frameborder可定义内网页的边框

使用 iframe 作为链接的目标

<iframe src="URL" name="iframe_a"></iframe>
<p><a href="url" target="iframe_a">链接到内网页</a></p>
