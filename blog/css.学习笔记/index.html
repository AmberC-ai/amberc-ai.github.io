background-position

background-position属性改变图像在背景中的位置，有三种定位方式

关键字

定位背景图片位置的关键字包括center、top、bottom、left、right.关键字出现的顺序不作要求，但不能超过两个关键字，如果只出现一个关键字，则另一关键字默认为center.如top left将图片放在元素内的左上角，right将背景图片放在元素右边中部。

p{
background-image:url('...');
background-position:top right
}

百分数值

也可使用百分数值对背景图片进行定位。如60%，40%即是指元素内定位为60%，40%的点与图片内定位为60%，40%的点重合。再如，0%，0%即是指图片的左上角与元素的左上角重合，50%，50%指元素中心与图片中心重合，即图片居中放置。如果只有一个百分数，则该百分数表示水平方向，垂直方向默认为50%.

p{
background-image:url('...');
background-position:66% 33%;
}

长度值

长度值描述的是图片左上角相对于元素内边距左上角的偏移。如50px，100px表示图片左上角距元素内边距右边50px，下边100px.

p{
background-image:url('...');
background-position:50px 100px;
}

background-attachment

将background-attachment属性设置为scroll（滚动）或fixed（固定）可以决定背景图片是否随文档的滚动而滚动。默认值为scroll，设置为fixed后图片在可是区域内是固定不动的，这能够避免因滚动而造成图片消失或位置改变的情况。

p{
background-image:url('...');
background-attachment:fixed;
}

background-repeat

如果需要背景图片在页面平铺可设置background-repeat属性，repeat-x和repeat-y分别对应水平方向上和垂直方向上的重复，no-repeat则不允许重复。背景图片默认从元素的左上角开始。

p{
background-image:url('...');
background-repeat:repeat-x;
}

文本缩进

css提供了text-indent属性方便实现文本缩进。该属性一般只对块级元素有作用，若想对行内元素进行文本缩进，可设置内边距。缩进的长度可以用长度值和百分数表示，用长度值表示时可以用负值,百分数值表示缩进长度是父元素长度的百分比。

p{
text-indent:5em;
}

p{
text-indent:-10em;
}

p{
text-indent:10%;
}

字母间隔和字间隔

letter-spacing和word-spacing属性分别设置字母间距和字符间距。可以将字母或字符的间隔扩大或缩小。这两个属性的用法基本相同。

p.letter{letter-spacing：20px}
p.word{word-spacing:-0.5em}

字体系列

用font-family属性定义文本的字体系列，css字体系列分两种：

通用字体 Serif
Sans-serif
Monospace
Cursive
Fantacy
特定字体 Times...

一般来说，设置的字体系列里都会有多种特定字体或通用字体系列，用来防止用户代理因没有安装某种特定字体而使用默认字体所带来的的显示效果差别太大的情况。

p {font-family: Times, TimesNR, 'New Century Schoolbook',Georgia, 'New York', serif;}

上面的代码设置的字体会按照前后顺序依次检索，直到检索到用户代理已经安装的字体。

注：当字体名称中含有空格或#，￥等字符时，要用单引号或双引号括起来

字体加粗

font-weight设置文本的粗细

关键字：

  • normal:正常显示文本
  • bold：可将字体设置为粗体
  • bolder：从继承值上移加粗度
  • lighter：从继承值下移加粗度

数值：

用100~900的整百数可表示九级加粗度，400等价于normal,700等价于bold

p.thick {font-weight:bold;}
p.thicker {font-weight:900;}

字体大小

font-size属性用来设置字体大小

用像素设置

用像素设置的是绝对字体大小

用这种方法设置的字体大小不会随浏览器的改变而改变

h1 {font-size:60px;}

用em设置

用em设置的是相对字体大小

用这种方法设置的字体大小会随父元素字体大小和浏览器改变而改变

h1 {font-size:3.75em;}

用百分数和em设置

在所有浏览器中均有效的方案是为 body 元素（父元素）以百分比设置默认的 font-size 值

body {font-size:100%;}
h1 {font-size:3.75em;}

链接

能够为链接设置样式的属性有很多种，如color,background,font-family等

可以为链接的不同状态分别设置样式

链接有四种状态：

  • a:link-未被访问的链接
  • a:visited-已经访问的连接
  • a:hover-鼠标指针位于链接的上方

  • a:active-链接被点击的瞬间

    a:link {color:#B2FF99;}
a:visited {text-decoration:underline;}
a:hover {background-color:#FF704D;}
a:active {font-family:Verdana, Arial, Helvetica, sans-serif;}

注：链接样式必须按照a:link,a:visited,a:hover,a:active的顺序设置

外边距同样遵循值复制原则

外边距合并

当两个垂直外边距相遇时，它们将形成一个外边距。合并后的外边距的高度等于两个发生合并的外边距的高度中的较大者。

当一个元素包含在另一个元素中时（假设没有内边距或边框把外边距分隔开），它们的上和/或下外边距也会发生合并

一个空元素，它有外边距，但是没有边框或填充。在这种情况下，上外边距与下外边距就碰到了一起，它们会发生合并

行内框、浮动框或绝对定位之间的外边距不会合并

框模型

背景应用于由内容和内边距、边框组成的区域

在 CSS 中，width 和 height 指的是内容区域的宽度和高度

选择器

元素选择器和类选择器

这两个选择器对某一个或某一类元素进行选择

h1 {color:blue;}
h2 {color:silver;}

选择器分组

若希望对多个或多类元素进行相同的设置，可以用选择器分组使她们有相同的属性

body, h2, p, table, th, td, pre, strong, em {color:gray;}
通配符选择器

当想要使所有元素拥有某种相同的值，可用*{…}声明， *代表选择所有元素

* {color:red;}
声明分组

当想要声明多种属性时，用；将各个声明隔开

h1 {
  font: 28px Verdana;
  color: blue;
  background: red;
  }

类选择器

  • 在使用类选择器之前首先要用class属性为元素归类

  • 在类别名前加上 . 就可选择该类元素

    <style type="text/css">
.important {color:red;}
</style>
</head>

<body>
<h1 class="important">This heading is very important.</h1>

<p class="important">This paragraph is very important.</p>
结合元素选择器
p.important {color:red;}

此时选择器会匹配class属性为important的p元素

多类选择器

当一个元素的class属性有多个值时，选择了其中任意一个或多个值且未选择除该元素class值以外的值的选择器都会对该元素起作用，但选择了多个值的选择器只对含有所有这些值的元素起作用。

.important {font-weight:bold;}
.warning {font-style:italic;}
.important.warning {background:silver;}
.important.urgent {background:silver;}
<p class="important">
This paragraph is a very important warning.
</p>
<p class="important warning">
This paragraph is a very important warning.
</p>

前三个选择器均对

元素起作用，第三个选择器只对第二个

元素起作用，第四个选择器对两个

元素都不起作用

ID选择器

  • 在使用ID选择器之前首先要用id属性为元素命名

  • 在id名前加上 # 就可选择该类元素

    #intro {font-weight:bold;}
...
<p id="intro">This is a paragraph of introduction.</p>

  • 在同一个文档中，id选择器只能用一次

  • id的值中不能有词列表，也不能有空格符

  • 类选择器和ID选择器都要区分大小写

属性选择器

选择器 描述
[attribute] 用于选取带有指定属性的元素

选择所有含title属性的元素

*[title] {color:red;}

选择含href属性的a元素

a[href] {color:red;}

选择既含href又含title的a元素

a[href][title] {color:red;}

后代选择器

后代选择器可以将嵌套在某一特定元素的元素选择出来，使选择器后面的声明只对某一元素下的某一特定元素起作用，无论该元素嵌套得有多深。

h1 em {color:red;}

子元素选择器

与后代元素选择器不同，子元素选择器只选择某元素的子元素，而不会对嵌套得更深的元素进行选择

h1 > strong {color:red;}

相邻兄弟选择器

相邻兄弟选择器可选择紧接在某一元素后面的选择器，这两个选择器必须有同一个父元素

h1 + p {margin-top:50px;}

子元素选择器中的>和相邻兄弟选择器中的+两边都可以有空格
