HTML学习笔记 HTML简介 HTML是一种超文本标记语言，使用标记标签来描述网页。 一般使用专业的HTML编辑器来编辑HTML，主要包括： Adobe Dreamweaver Microsoft Expression Web CoffeCup HTML Editor 也可使用文本编辑器如：Notepad(PC)、TextEdit(Mac)来学习HTML HTML基本格式 <!DOCTYPE HTML> <html> <head> <meta> <title></title> </head> <body> <h1>第一个标题</h1> <p>第一个段落</p> </body> <html> !DOCTYPE用来声明html的版本类型，不同版本的html有不同的声明，详情见百度。元素中的内容为页面可见内容 属性 属性提供了有关HTML元素的更多信息，总是在开始标签中规定。如： <html> <body> <div name="Amber"> 块 </div> </body> </html> name属性为元素定义名称 注：属性总是以名称/值对的形式出现，如name=Amber,name是属性，Amber是属性值。 属性值始终被包含在引号内，通常为双引号。但有时属性值里本身就包含双引号，这时必须使用单引号。如： name='Bill"HelloWorld"Gates' 更多属性标签详见https://www.w3school.com.cn/tags/html_ref_standardattributes.asp HTML标题 HTML标题（heading)是通过h1-h6等标签进行定义的 <h1>这是一个标题</h1> <h2>这是一个标题</h2> <h3>这是一个标题</h3> 效果 这是一个标题 这是一个标题 这是一个标题 搜索引擎将根据标题来为网页的结构和内容编制索引，所以不要仅仅为了使用大号或粗体字而使用标题，而应用标题来呈现文档结构 HTML段落 HTML段落通过p标签进行定义 <p> 这是第一个段落 </p> <p> 这是第二个段落 </p> 效果
